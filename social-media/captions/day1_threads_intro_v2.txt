我叫 Martin，我正在做一件可能好傻嘅事。

我辭咗份全職工，一個人寫緊一套系統，想幫香港嘅中小企老闆慳返啲時間。

點解？因為我有個好朋友，佢喺觀塘開咗間小店。每次約佢食飯，佢都話「唔得閒」。唔係因為生意太好，係因為收咗舖之後仲要對數、排更、追貨。

有一次佢同我講：「我開舖係想做自己鍾意嘅嘢，點知而家大部分時間都係做 admin。」

嗰句說話一直留喺我心入面。

我開始留意，發現唔止佢一個係咁。好多香港小店老闆都面對同一個問題 — 用緊 3、4 套唔同嘅系統，每套都要錢，但佢哋之間完全唔 talk to each other。

所以我決定辭工，由零開始寫一套叫 Cardlink 嘅系統。名片、POS、庫存、會計、HR — 全部放埋一齊，為香港嘅小店而設。

我知呢個目標好大。老實講，有時夜晚對住個 mon 都會諗「係咪太天真」。

但每次諗起朋友嗰句「我只係想做自己鍾意嘅嘢」，就覺得值得繼續。

由今日開始，我會喺呢度分享成個過程 — 寫到 code 嘅開心、debug 到想死嘅崩潰、同每一個令我堅持落去嘅故事。

如果你都係一個香港小店老闆，我想聽你嘅故事。你當初點解會開舖？而家最辛苦嘅係咩？

留言話我知，或者 DM 我都得。我真係想聽。

因為 Cardlink 唔係我一個人嘅 project — 佢應該係我哋一齊 build 嘅嘢。
